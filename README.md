# Apocalyptic Dominion - Full Expanded Version

Complete base ruleset for Unciv with post-apocalyptic survival, domination, and breeding themes.

## Features
- 4 Civilizations with unique leaders and playstyles
- 12+ Units including advanced and great person equivalents
- 12+ Buildings including national wonders and mega-structures
- 9 Thematic Technologies
- 7 Powerful Policies with synergies
- Expanded Promotions and Tile Improvements
- Rich flavor text and adult-themed mechanics (abstracted)

## Installation
1. Download from this repository URL in Unciv.
2. Select as Base Ruleset when starting a new game.
